# Troubleshooting (Overview)

Terkadang konfigurasi antara sistem klien dan pusat penyedia identitas (NexaID) tidak langsung berjalan mulus. Halaman Troubleshooting ini kami susun sebagai pusat rujukan utama (*knowledge base*) bagi Admin maupun Developer dalam menangani anomali keamanan, penolakan koneksi, atau cacat sinkronisasi.

Silakan pilih kategori dari kumpulan masalah Anda:

- **[Masalah Seputar SSO (SSO Errors)](./sso-errors)**
  Berkaitan dengan kegagalan *redirect*, Callback URL yang ditolak, gagal *login*, atau ditolak masuk oleh sistem otentikasi.

- **[Masalah Seputar Token (Token Errors)](./token-errors)**
  Berkaitan dengan API yang mengembalikan HTTP `401 Unauthorized`, `Token Expired`, masalah *Header Authorization*, dan *JWT signatures*.

- **[Masalah Umum Lainnya (Common Issues)](./common-issues)**
  Berkaitan dengan konektivitas lintas server (IP Whitelist), hingga data pengguna/role yang seolah "hilang" atau menolak sinkronisasi ke aplikasi klien.
